# TempleDB Assets

This directory contains vector art and branding assets for TempleDB.

## Files

### Logos & Icons

- **logo.svg** - Main TempleDB logo (200x200)
  - Combines temple architecture with database imagery
  - Use for documentation, presentations, etc.

- **icon-small.svg** - Small icon version (64x64)
  - Optimized for CLI tools, favicons, and small displays
  - Simplified version of the main logo

### Banners

- **banner.svg** - Repository banner (800x200)
  - For README headers, documentation sites
  - Includes project name, tagline, and Terry Davis tribute

### Special Art

- **templeos-tribute.svg** - TempleOS-inspired art (640x480)
  - Uses the classic 16-color VGA palette
  - ASCII-art style honoring Terry Davis and TempleOS
  - Great for presentations about the project's inspiration

## Design Elements

### Color Palette

- **Primary**: `#00d4ff` (Cyan) - Data/Technology
- **Secondary**: `#e94560` (Red/Pink) - Temple/Structure
- **Background**: `#0a0e27`, `#1a1a2e` (Dark blues)
- **Accent**: `#0f3460` (Deep blue)

### TempleOS 16-Color Palette

The templeos-tribute.svg uses the authentic VGA 16-color palette that Terry Davis used in TempleOS:

- Black (#000000), Blue (#0000AA), Green (#00AA00), Cyan (#00AAAA)
- Red (#AA0000), Magenta (#AA00AA), Brown (#AA5500), Light Gray (#AAAAAA)
- Dark Gray (#555555), Light Blue (#5555FF), Light Green (#55FF55), Light Cyan (#55FFFF)
- Light Red (#FF5555), Light Magenta (#FF55FF), Yellow (#FFFF55), White (#FFFFFF)

### Typography

All designs use monospace fonts to reflect the CLI/database nature of the project and pay homage to TempleOS.

## Usage

All SVG files can be:
- Scaled to any size without quality loss
- Converted to PNG/other formats as needed
- Embedded directly in HTML/Markdown
- Used in presentations and documentation

### Example: Using in README

```markdown
![TempleDB](assets/banner.svg)

![Icon](assets/icon-small.svg)
```
